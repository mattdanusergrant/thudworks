How the code works

play(instrument, pattern, opts?) adds a part. tempo(bpm) and swing(0–70) set the clock. Every part loops to fill the longest one.

Patterns read one 16th-note per cell:

  • Drums — one character per step: x = hit, . = rest. Spaces are ignored, so you can group beats: "x... x... x... x..."
  • Pitched — space-separated tokens: a note like C4 / F#3 / Bb2 plays it, . = rest, - = hold (tie).

Helpers: rep(pattern, n) repeats a pattern n times · euclid(hits, steps[, rotate]) makes an evenly-spread drum pattern.

opts: { wave, cutoff, detune, gain } — e.g. { wave:'square', cutoff:1200, gain:0.4 } (pitched parts).

Instruments: kick, snare, clap, hat, openhat, cowbell, clave, tom · pitched: bass (808), synth, lead, pad, pluck.

Every sound is synthesized live in your browser — no samples, no licensing. Edit the code, hit Play. Pick a song above to start.
